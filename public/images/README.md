# Images Directory

## Required Images

### About Page
- `alex.jpg` - Profile photo for Alex Fasih (recommended size: 400x400px or larger)

### Blog Images
Place blog thumbnail images in the `blogs/` subdirectory:
- `/images/blogs/fp-a-future.jpg`
- `/images/blogs/startup-budgeting.jpg`
- `/images/blogs/data-visualization.jpg`

## Placeholder Images

If images are not available, the components will automatically use placeholder SVGs. However, you should replace these with actual images for production.

## Image Recommendations

- **Format**: JPG or PNG
- **Profile Photo**: Square (1:1 aspect ratio), at least 400x400px
- **Blog Thumbnails**: 16:9 aspect ratio, recommended 1200x675px
- **Optimization**: Compress images before adding to reduce page load time
